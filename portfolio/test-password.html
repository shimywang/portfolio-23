Launch: Game inspired by Aalto-1 Satellite Project

Product design, Game design, Prototyping

10.2019 - 12.2019

Espoo, Finland

Course Project

This is a course project for designing a game to be sold by the Aalto University Shop inspired by the research and work developed around the project Aalto 1, responsible for launching the first Finnish satellite into space.

Collaborated with Pedro Garcia Alcazar, Yi-Tsu Hsiao, Leena Nurmi

1 Inspiration

We started by researching on the story of Aalto-1 Project and existing game.

BURRO- Exisiting game

BURRO , also have many other names in different cultures, is a popular party game that is suitable for most of the age groups in different user scenarios.
BURRO has a set of game rules that has been proven to work perfectly and being loved by players.

Tag represneting launching satellites

Aalto-1 Project has developed their own souvenirs. One of their souvenirs is a fabric tag with text “Remove beofore launch”.
We are inspired for using this tag to represent the process of launching satellites.

Key steps of Aalto-1 development process

Aalto-1 is the first FInnish lanunched satellite. We wants to share the key steps of how the satellite is being biult, to achieve Aalto Shop’s main value of bring out product with educational purpose.

Exploration on graphics of cards

We explored different graphic design of the cards.

3 Final Design

Target Group & Type of game

The game is designed for tourists visiting Aalto who wants to get souvenirs from Aalto shop, therefore, it is small in size and easy to be carried away, not very expensive.

3-8 players with family or friends (7+ yo)

5 - 10 mins

What is inside a game package

In every game package, there are 4*8 = 32 cards, and 3* 8 = 24 tags.
Every card represent a key step of Aalto-1 Project.

Tags are made out of recycled fabrics we found in workshops in Aalto, which makes them unique for every single product. We designed a tag container for holding these tags. The tag container has exact same shape as our cards, and therefore can be easily collected together.

Package Design

The package of the game is also made from ply wood, for the purpose of remaining simplicity while providing a satellite-like shape of packaging as well as attracting customers while they visiting the shop. Because the tag container has exactly same shape as cards, they can be easily collected together with a simple wood board.

Also, this ply wood package can also help the rest cards standing alone when playing game with less then 8 people, which is also easy to manipulate when putting cards back.

We also designed another packaging, that is also made out of recycled materials from Aalto’s student workshops and is more traditional.
